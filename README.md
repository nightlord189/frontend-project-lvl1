## Description
Brain Games is training JS-project, consists of five console brain games: even, calculator, gcd, progression, prime numbers


## Getting Started
### Setup:
```
git clone https://github.com/nightlord189/frontend-project-lvl1
npm install
npm link
```

### Launch:
```
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
